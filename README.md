# Portfolio — HTML & CSS files

## Overview
Brief descriptions of the HTML and CSS files contained in the `portfolio` directory. Each HTML file is a page that links to every other page, along with the CSS files for consistent styling (header, footer, color scheme, etc.) and responsive behavior.

## Files (HTML and CSS)
- `index.html`  
    Home / landing page. Contains professional portrait and brief intro, and footer. Links to other files in portfolio.

- `contact.html`  
    Contact form and social links. Form fields (name, email, message) and link targets for email/social profiles. Form may use client-side validation.

- `about.html`  
    Personal bio and skills, including a profile photo.

- `resume.html`  
    Display of what a résumé of mine would look like. *It is important to note that it is not a real résumé.*

- `courses.html`  
    Table of courses within selected major, including course descriptions.

- `projects.html`  
    Project gallery with titles, short descriptions, and source documents.

- `styles.css`  
    Primary stylesheet: color variables, layout, and styles for header/footer, buttons, and forms. Includes CSS custom properties and component classes.

## Structure & Conventions
- Semantic HTML (header, nav, main, footer).  
- Use CSS variables for colors and spacing for easy theming.  
- Keep layout and components reusable across pages.

## How to preview
Open `index.html` (or any HTML file) in a browser. Every HTML file can be used to navigate to every other HTML file.
